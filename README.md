
# Doodle Jump

HTML CSS Canvas and TypeScript implementation for Doodle Jump game

## Run locally


1. clone the repo by

```
https://github.com/BikramNischal/doodle-jump.git
```

2. change to project directory
```
cd doodle-jump
```

3. install dependencies
```
npm install 
```

4. build the project
```
npm run build
```

5. run the project
```
npm run dev
```
